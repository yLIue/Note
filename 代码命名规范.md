## 一、 常用的命名方法

### 1.下划线命名

字母全部小写,不同单词用下划线分开

如 `user_name`

### 2.小驼峰命名

首单词小写,其余大写

如 `userName`

### 3.大驼峰命名

单词字母大写

如 `UserName`

## 二、 变量命名规范

采用变量解释加变量名的命名方式,小驼峰命名

类型可有可无
### 1. 常用的变量解释

| 类型          | 描述 |
| ------------- | ---- |
| int, float... | 类型 |
| count         | 计数 |
| ing           | 当前 |
| new           | 新的 |
| index         | 索引 |
| wait          | 等待 |
| end           | 结束 |
| len           | 长度 |
| max           | 最大 |
| min           | 最小 |
| before        | 之前 |



## 三、 函数编写规范

### 1.函数命名规范

#### 1.1 正常命名

采用前缀加函数名

前缀可有可无

#### 1.2 特殊命名

| 写法       | 描述         |
| ---------- | ------------ |
| `B`From`A` | 从A里面获得B |



#### 1.3 函数前缀

##### 命名规范

全部小写,可以没有

##### 常用前缀

| 前缀   | 常用的用途 |
| ------ | ---------- |
| get    | 获取       |
| check  | 检查/判定  |

### 2.函数内部变量命名规范

和变量命名规范一致,内部变量需要加上下划线
