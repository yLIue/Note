## 目录



## 命名规范

- 只能由字母,数字,下划线构成

- 不能数字开头

## 变量类型

| 变量类型     | 描述          | 输出 | 字节x64 |
| ------------ | ------------- | ---- | ------- |
| int          | 整型          | %d   | 4       |
| short        | 短整型        |      | 2       |
| long         | 长整型        | %ld  | 4_8     |
| long long    | 超长\长长整型 | %lld | 8       |
| unsigned int | 无符号整型    | %u   |         |
| float        | 浮点型        | %f   | 4       |
| double       | 双精度浮点型  | %lf  | 8       |
| char         | 字符型        | %c   | 1       |
| *            | 指针          | %p   |         |
|              | 字符串        | %s   |         |
| #define      | 全局变量      |      |         |

## 输出格式

| 用法    | 描述                     | 备注              |
| ------- | ------------------------ | ----------------- |
| %.`num` | 省略小数点后`num`位数    | 奇进偶退 四舍五入 |
| %`num`  | 该输出占用多少个字符位置 |                   |

## 特性

- 关于强制类型转化
  - 字符转整型	结果为ASCII值
  - 浮点转整型	结果忽略小数点

## 指针

### 结构体指针赋值

创建一个结构体Test

```c
typedef struct Test{
    int _number;
} Test;
```

创建一个指针并指向它

```c
Test temp;
Test *p = &temp;
```

地址指向,是地址去指向,不是指针

```c
p -> _number = 5;
```
